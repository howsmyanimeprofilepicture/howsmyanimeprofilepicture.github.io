분류모델에 대한 단상

분류모델 그리고 다항분포

  • 문득 든 생각인데 분류모델이 최종적으로 내놓는 것은 다항분포가 아닌가? 즉 입력으로 넣어준 무언가가 어떤 다항분포를 따르는지에 대해 리턴하는 게 바로 분류모델이라고 생각한다. (정확히는 카테고리 분포이긴함)

  • 예를 들어 댕댕이 사진을 분류모델에 인풋으로 넣었다.

  •

    input = [
        [0, 0, 10, 0, 0, 70],

    #대충이게 고양이 사진의 매트릭스라고 치자... ;ㅈㅅ;

  • 분류모델이 이진분류라면 댕댕이인지 아닌지에 대한 이항분포를 반환할 것이고, (엄밀히는 베르누이분포임)

    • 이항분포를 B(n, p)라고 표현한다면, n은 1이 될 것이고, p는 분류모델이 판단한 인풋 이미지가 댕댕이일 가능성이 될 것이다.
    • 즉, 분류 모델은 댕댕이 사진이 댕댕이인지 아닌지에 대해 확인하는 1회의 시행에서 댕댕이가 될 확률을 p로 판단한 것이다.
    • 분류모델이 (0.7, 0.3)을 리턴했다면, 분류모델은 해당 사진이 댕댕이일 확률을 0.7로 판단했다고 이해할 수 있다. (첫번째 원소가 댕댕이일 확률, 2번째 원소가 아닐 확률)

  • 분류모델이 댕댕이, 야옹이, 비둘기, 개구기, 그 외에 대한 다중분류라면 다항분포를 반환할 것이다. (이것 역시 엄밀히는 카테고리분포임)

    • 모델이 반환하는 다항분포를 Mul(n, p)로 표현한다면, n은 1이 될것이고,

    • p는 모든 원소의 합이 1인 5차원 벡터가 될 것이다.

      • p = (p1, p2, ... , p5)
      • p1은 1번의 시행(이미지 분류)의 결과로 이미지가 댕댕이일 확률,
      • p2은 야옹이일 확률,
      • 순서대로 비둘기, 개구리, 그 외라고 하자.
      • p1 + p2 + ... + p5 = 1 이 성립한다.

    • 모델이 만약 (0.9, 0.05, 0.01, 0.01, 0.03)을 리턴했다면

      • 댕댕이일 확률 90%, 야옹이일 확률 5%, 비둘기 1%, 개구리 1%, 그외 3% 라고 판단한 것이다.
      • 입력값으로 전달된 사진이 Mul(1; 0.9, 0.05, 0.01, 0.01, 0.03)의 다항분포를 따른다고 판단한 것과 같다.

 

 

 

분류모델이 리턴한 예측확률 과 실제확률, 그리고 엔트로피

댕댕이 사진의 실제 확률 분포는 Mul(1; 1, 0, 0, 0, 0)이다.

100% 댕댕이라는 것이다.

엔트로피는 0이 된다.

 

만약 분류모델이 예측한 확률분포에는 어느 정도 엔트로피가 끼어 있다.

확신이 아닌 확률적으로 판단했기 때문이다.

정보 엔트로피는 시행(혹은 정보)의 불확실성을 나타내는 척도이다.

 

😉엔트로피 계산법

H(X) = -(p1*log2p1 + p2*log2p2 + ... + pn*log2pn)

p1 + p2 + ... + pn = 1

어떤 시행의 독립적인 결과가 총 n가지가 있고,

각 결과들이 나올 확률을 p1, p2, ... , pn이라 할 때,

해당 시행의 엔트로피는 위와 같이 계산한다.

log의 밑을 2로하면 섀넌, 비트 단위를 붙여 표현하고,

자연로그의 밑(e)를 밑으로하면 nat라고 표현한다.

 

 

 

 

참고) 크로스 엔트로피, KL Divergence

 

H(P, Q) = -(p1*log2q1 + p2*log2q2 + ... + pn*log2qn)

pi는 실제확률, qj는 예측확률이다.

 

DKL(P||Q) = H(P, Q) - H(P) = 교차엔트로피 - P의 엔트로피
