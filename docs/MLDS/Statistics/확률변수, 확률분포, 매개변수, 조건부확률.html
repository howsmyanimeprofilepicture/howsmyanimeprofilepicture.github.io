확률변수, random variable

  • 확률변수는 함수다.

    • 확률론에서, 확률 변수(random variable)는 확률 공간에서 다른 가측 공간으로 가는 가측 함수이다.
    • 표본공간(sample space)의 각각의 원소들을 실수에 대응시키는 함수이다.
    • 조금 더 쉽게 이해하자면 표본공간에 있는 원소들, 즉 어떠한 시행의 결과들을 실수에 대입시킨 것으로 이해하면 될 것이다.
    • 밑의 예시를 참고하는 것이 보다 이해에 도움이 될 것 같다.

  • 예시

    • 동전 2회 던지기

    • 표본 공간(sample space) = { (H, H), (H, T), (T, H), (T, T) }

    • Head가 나온 수를 확률변수로 한다고 하면

    • 확률 변수 : X = {0, 1, 2} 가 된다.

    • 표본 공간의 (H, H)는 2에 대응될 것이고,

    • (H, T)와 (T, H)는 1에 대응될 것이다.

    • (T, T)는 0에 대응될 것이다.

    • 동전을 2번 던졌을 때 Head가 나온 수를 확률변수로 하면

    • 확률변수는 {0, 1, 2}가 된다.

 

 

확률분포, probability distribution

  • 확률 분포(probability distribution)는 확률변수가 특정한 값을 가질 확률을 나타내는 함수이다.

    • 즉 확률분포는 확률변수를 변수로 갖는 함수이다.

  • 위의 동전 2번 던졌을 때 Head가 나온 수를 확률변수로 하면

    • P(0) = 1/4
    • P(1) = 1/2
    • P(2) = 1/4

  • 연속확률분포의 경우 -∞에서 ∞까지 정적분하면 1이 된다.

  • 이산확률분포의 경우 모든 확률변수에 대한 P(X)값을 더하면 1이 된다.

변수(variables)와 매개변수(parameters)

  • 위 도표는 μ와 σ에 따른 정규분포의 모양이다.

 

  • 정규분포의 확률밀도함수는 위와 같다.
  • 매개변수는 모수라고도 한다.
  • 위 확률밀도 함수에서 변수는 x이고,
  • 매개변수(parameters)는 μ와 σ이다.
  • 변수와 매개변수의 차이를 말로 설명하는 것이 쉽지 않은데
  • 일단 함수란 것은 변수에 따라 함수값이 결정된다. 변수는 말 그대로 변하는 수이다.
  • 매개변수는 일종의 가정 같은 것으로 이해하면 좋을 것 같다.
  • 함수의 그래프를 그릴 때 변수는 하나의 축을 형성하지만, 매개변수는 축을 형성하지 않는다.
  • 왜냐하면 함수는 변수에 따른 함수값(y)을 표시하는 것이 목적이지, 매개변수에 따른 함수값을 표시하는 것이 목적이 아니기 때문이다.
  • 결국 매개변수와 변수는 함수의 목적에 따라 달라지기도 한다는 것이다.
  • 확률밀도함수는 확률변수 x에 따른 확률값을 확인하는 것이 목적이기 때문에 확률변수(x)가 변수가 되고, 평균(μ)과 분산(σ)은 매개변수가 된다.
  • 하지만 가능도함수의 경우 모집단의 매개변수에 따른 가능도를 측정하는 것이 목적이다. 이로 인해 정규분포를 따르는 모집단에서 추출한 표본에 대한 가능도 함수에선 μ와 σ이 변수가 된다.

 

  • function_name(variables ; parameters)로 표기한다.
  • 정규분포의 경우 확률변수(x)가 변수고, 평균(μ)과 표준편차(σ)가 매개변수이기 때문에 f(x; μ, σ)로 표기할 수 있다.

 

 

매개변수(媒介變數), 파라미터(parameter), 모수(母數)는 수학과 통계학에서 어떠한 시스템이나 함수의 특정한 성질을 나타내는 변수를 말한다. 일반적으로는 θ라고 표현되며, 다른 표시는 각각 독특한 뜻을 지닌다.
