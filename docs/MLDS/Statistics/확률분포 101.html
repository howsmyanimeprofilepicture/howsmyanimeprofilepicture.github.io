이항분포

이항분포 이해하기

 

  • 이항분포는 시행 횟수 n과 각 시행의 성공 확률 p를 매개변수로 갖는다.
  • 이항분포에서의 각 시행은 성공 혹은 실패만의 결과를 갖는 베르누이 시행이다.
  • 시행 횟수 n이 1인 이항분포는 베르누이 분포이다.

 

확률 질량 함수

  • 확률변수 X가 매개변수로 n과 p를 갖는 이항분포를 따른다면 X ~ B(n, p)라고 표현한다.

  • n번의 시행횟수를 갖고, 각 시행의 성공률로 p를 갖는 이항분포의 확률질량함수는 f(x; n, p)로 표현할 수 있다. (변수 x는 총 n번의 시행 중 성공한 시행의 수이다.)

    • f(x; n, p) = nCx * px (1-p)n-x

    • nCx = n! / x!(n-x)!

  • nCx는 이항 계수이다.

 

이항 계수

  • 이항 계수 nCr은 (a+b)n 에서 arbn-r의 계수를 의미한다.

    • nCx = n! / x!(n-x)!

  • 근데 왜 이항 계수를 이항분포에서 사용하는 걸까?

  • 이항 계수를 (성공 + 실패)n이라고 생각해보면 이해가 쉬울 것이다.

  • 이항분포에서 f(3)은 성공 횟수가 3인 경우의 수를 찾는 것이다.

  • 즉 성공3실패n-3의 계수를 찾는 것이다.

  • 그래서 f(x; n, p) = nCx * px (1-p)n-x 꼴로 확률질량함수를 나타낼 수 있게 되는 것이다.

 

 

평균과 분산

  • 만약 확률변수 X가 X ~ B(n, p)를 따른다면,

  • E(X) = np

  • Var(X) = np(1 - p)
