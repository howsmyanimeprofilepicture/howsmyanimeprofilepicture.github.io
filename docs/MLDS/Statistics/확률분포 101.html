이항분포

이항분포 이해하기

 

  • 이항분포는 시행 횟수 n과 각 시행의 성공 확률 p를 매개변수로 갖는다.
  • 이항분포에서의 각 시행은 성공 혹은 실패만의 결과를 갖는 베르누이 시행이다.
  • 시행 횟수 n이 1인 이항분포는 베르누이 분포이다.

 

확률 질량 함수

  • 확률변수 X가 매개변수로 n과 p를 갖는 이항분포를 따른다면 X ~ B(n, p)라고 표현한다.

  • n번의 시행횟수를 갖고, 각 시행의 성공률로 p를 갖는 이항분포의 확률질량함수는 f(x; n, p)로 표현할 수 있다. (변수 x는 총 n번의 시행 중 성공한 시행의 수이다.)

    • f(x; n, p) = nCx * px (1-p)n-x

    • nCx = n! / x!(n-x)!

  • nCx는 이항 계수이다.

 

이항 계수

  • 이항 계수 nCr은 (a+b)n 에서 arbn-r의 계수를 의미한다.

    • nCx = n! / x!(n-x)!

  • 근데 왜 이항 계수를 이항분포에서 사용하는 걸까?

  • 이항 계수를 (성공 + 실패)n이라고 생각해보면 이해가 쉬울 것이다.

  • 이항분포에서 f(3)은 성공 횟수가 3인 경우의 수를 찾는 것이다.

  • 즉 성공3실패n-3의 계수를 찾는 것이다.

  • 그래서 f(x; n, p) = nCx * px (1-p)n-x 꼴로 확률질량함수를 나타낼 수 있게 되는 것이다.

 

 

평균과 분산

  • 만약 확률변수 X가 X ~ B(n, p)를 따른다면,

  • E(X) = np

  • Var(X) = np(1 - p)

  • 계산하면 실제로 저렇게 나온다. 유도와 증명은 아래 위키피디아 문서를 참고하기 바람

    • 이항분포의 평균과 분산의 명확한 유도법

다항분포

  • 시행의 결과가 여러가지이다. (베르누이 시행과의 차이다.)

  • 시행의 결과는 결과1, 결과2, ... , 결과k가 있다.

  • 시행의 각 결과의 확률은 p1, p2, ... , pk라고 할 수 있다.

    • p1 + p2 + ... + pk = 1 이 성립한다. 당연하다.

  • 그리고 해당 시행을 n번 반복했을 때 각 결과가 발행한 횟수를 x1, x2, ... , xk라고 하자.

    • x1, x2, ... , xk = n 이 성립한다. 이것도 당연하다.

  • 위와 같은 시행을 n번 반복했을 때를 확률질량함수로 표현하면

    p(x1, x2, ... , xk ; n ; p1, p2, ... , pk) = {n!/(x1! x2! ... xk!)}p1x1 p2x2... pkxk

    와 같이 나타낼 수 있다.

    벡터로 표현하면?

    x=(x1, x2, ... , xk)
    θ=(p1, p2, ... , pk)
    p( x ; n ; θ) = {n!/(x1! x2! ... xk!)}p1x1 p2x2... pkxk

 

 

  • 주사위를 2번 던진 결과를 다항분포의 확률질량함수로 표현해보자

    x=(x1, x2, x3, x4, x5, x6)
    θ =( (1/6), (1/6), (1/6), (1/6), (1/6), (1/6) )
    p(x; n=2; θ )
    • 참고로 0!=1이다.
    • xk는 n=2번의 시행에서 그 시행의 결과로 주사위k눈이 나온 횟수를 의미한다.
    • p(1, 0, 1, 0, 0, 0; 2)은 주사위를 2번 던져 1의 눈이 한 번 3의 눈이 1번 나올 확률을 뜻한다.
    • 그 값은 { 2! / (1! 1!) }*(1/6)*(1/6)가 되겠지?

 

다항 계수

  • 다항분포의 확률질량함수에 계수처럼 붙는 n!/(x1! x2! ... xk!)에 대해서는 위키피디아의 다항 계수 문서를 참고해보길 바란다.

  • 이항 계수와 크게 접근방법이 다르지 않다.

  • 결국 다항분포의 확률을 계산한다는 것은 (결과1 + 결과2 + ... + 결과k)n에서 (결과1)x1(결과2)x2... 결과k)xk의 계수가 몇인지를 찾는 것과 똑같다는 것이다.

    • (x1 + x2 + ... + xk = n)
    • 말로 설명이 잘 안되는데 한 번 천천히 생각해보라 이해가 될 것이다

 

 

평균과 분산

  • 시행의 결과 {R1, R2, ..., Rk}

  • 각 시행의 확률{p1, p2, ... , pn}

    • p1 + p2 + ... + pn = 1

  • 특정한 결과 Ri에 대한 이항분포로 만들 수 있다. Ri가 나왔는지 아닌지로 이분법적인 베르누이 시행으로 만들 수 있다는 소리다.

  • B(n, pi)로 말이다.

  • 이와 같은 이항분포를 따르는 확률변수 Xi가 있다고 치자. Xi ~ B(n, pi)

    • E(Xi) = npi

      • 확률변수 Xi는 n번의 시행의 결과로 Ri가 나온 횟수를 의미하며,
      • 따라서 Xi = {0, 1, 2, ..., n}이 된다.
      • E(Xi)은 n번의 시행에서 그 시행의 결과로 Ri가 평균적으로 몇 번 나오는지를 의미한다.
      • 동시에 E(Xi) 확률변수 Xi의 각 값들( 0, 1, 2, ... , n )을 각각의 확률로 가중평균한 값이다.

    • Var(Xi) = npi(1 - pi)

 

주사위를 예시로 들어보자

f(x; n=2 ; θ)
x = (x1, x2, x3, x4, x5, x6)
x1 + x2 + x3 + x4 + x5 + x6 = 2
θ = ((1/6), (1/6), (1/6), (1/6), (1/6), (1/6), )

시행은 두번이다. (n = 2)

E(X5) = np5 = 2*(1/6) = 1/3

(2번의 주사위 던지기에서의 5의 눈에 대한 기댓값)

즉, 2번의 주사위 던지기에서 5의 눈이 평균적으로 몇번 나올까를 의미한다.

 

다른 예시를 들면, "3번의 주사위 던지기에서 2혹은 6의 눈은 평균적으로 몇번 나올까?"도 생각해볼 수 있다.

B(n=3; (p2 + p6))

E(X2∪X3) = n(p2 + p6) = 3* (1/3) = 1이 된다.
