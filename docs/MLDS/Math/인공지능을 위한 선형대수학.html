선형 결합 Linear Combination

  • 선형 결합의 개념을 이해하는 것이 중요해보인다.

    • 벡터 v가 벡터 v1, v2, ..., vn의 선형결합이라는 것은
    • v = c1v1 + c2v2 + ... + cnvn을 만족하는
    • 실수 c1, c2, ..., cn가 존재한다는 것이다.

  • 예시

    • 3차원 벡터 (1, 1, 1)은 (1, 0, 0)과 (0, 1, 1)의 선형결합이다.

    • (2, 1, 1) 역시 (1, 0, 0)과 (0, 1, 1)의 선형결합이라 할 수 있다.

       

Span (선형 생성?)

벡터들에 대한 Span 집합을 구한다는 것은?

  • Span이란 벡터들의 선형결합으로 만들 수 있는 모든 벡터들의 집합이다.

 

  • Span {v1}은 직선이 된다.

    • 한 개의 벡터로 만들 수 있는 선형결합이란
    • 결국 그 하나의 벡터에 스칼라배를 해주는 것이 전부이기 때문이다.
    • 즉, 벡터를 늘리고 줄이고 반대로 늘리는 게 전부이기 때문이다.

     

  • Span {v1, v2}은 평면이 된다. (단, 두 벡터가 평행하지 않을 때)

Span이 부분공간이 된다는 것을 증명하자.

  • ℝn의 벡터 v1, 2, ... , vk에 대해 Span {v1, v2, ... , vk}는 ℝn의 부분공간이다.

    • ℝn은 n차원의 실수좌표공간(Real coordinate space)를 의미한다.
    • (사실 n차원 좌표 자체가 평행하지 않은 n개의 벡터로 공간을 표현하는 기술이다.)
    • (좌표공간와 벡터공간은 사실상 같은 개념 아닌가? 뇌피셜임)
    • 교재에 따라 Rn으로 표기하기도 한다.
    • ℝn은 벡터공간이다. 벡터공간의 9가지 조건이 모두 성립하기 때문이다.
    • span 역시 벡터 공간이다.

 

벡터공간, 부분공간

  • 벡터공간

    • 벡터를 다룰 수 있는 공간

    • 벡터끼리 더하고, 벡터에 스칼라배를 하는 것이 가능한 공간을 의미한다.

    • 이를 만족하기 위해서는 다음이 성립해야 한다.

      i) 덧셈에 대해 닫혀있다.

      ∀v1, v2 ∈ vn, v1+v2 ∈ vn

      ∀는 '임의의 모든 것에 대하여'라는 뜻이다.

      즉 vn의 임의의 벡터 v1, v2에 대하여 v1+v2 역시 vn의 원소라는 뜻이다. (닫혀 있다라는 소리임 ㅇㅋ?)

      ii) 덧셈의 교환법칙

      ∀v1, v2 ∈ vn, v1+v2 = v2+v1

      iii) 덧셈의 결합법칙

      ∀v1, v2, v3 ∈ vn, (v1v2)+v3=v1(v2+v3)

      iv) 덧셈의 항등원

      ∀v1 ∈ vn, ∃e ∈ vn s.t. v1+e = v1

      이때, e를 0(영벡터)이라고 쓴다.

      v) 덧셈의 역원

      ∀v1 ∈ vn, ∃x ∈ vn s.t. v1+x = 0

      이때, x를 -v1 이라고 쓴다.

      vi) 스칼라 배에 대해 닫혀있다.

      ∀k ∈ ℝ, ∀v ∈ vn, kv ∈ vn

      vii) 스칼라 배의 결합법칙

      ∀k1, k2 ∈ ℝ, ∀v ∈ vn, k1(k2v) = (k1k2)v

      viii) 스칼라 배의 항등원

      ∀v ∈ vn, 1 * v = v

      ix) 분배법칙

      ∀k1, k2 ∈ ℝ, ∀v1, v2 ∈ vn, (k1 + k2)(v1+v2) = k1v1 + k2v1 + k1v2 + k2v2

    • 위 9가지가 성립하는 집합 vn을 벡터공간이라고 한다.

    • 그리고 어떠한 집합이 위 9개의 법칙을 만족한다면, 그 집합은 벡터라고 생각해도 무방하다고 할 수 있다.

 

  • 부분공간 :

  • 벡터공간 vn의 부분집합 wn이 벡터공간이면,

  • wn을 vn의 부분공간이라고 한다.

  • 정리

    • vn : 벡터공간 ,
    • wn ⊆ vn , wn은 벡터 공간 vn의 부분집합이다.
    • v1, v2 ∈ wn : v1 + v2 ∈ vn
    • wn의 원소인 v1과 v2의 합은 벡터공간 vn의 원소가 된다.
    • 하지만 v1 + v2가 반드시 wn이 되는 것은 아니다.
    • 만약 wn 역시 벡터공간의 9가지 조건이 성립한다면, wn역시 일종의 벡터공간이며, vn의 부분공간이라 할 수 있다.

선형독립이란?

  • 벡터 v1, v2, v3 가 선형독립 관계에 있다함은 아래와 같이 표현할 수 있다.

    • v1은 v2와 v3의 선형결합으로 표현할 수 없고,

    • v2는 v1과 v3의 선형결합으로 표현할 수 없고,

    • v3 역시 v1과 v2의 선형결합으로 표현할 수 없다.

    • 이렇게 선형적으로 독립적인 관계를 선형독립이라 한다.

       

  • 수학적으로 표현하면 아래와 같다.

    • 집합 S = {v1, v2, v3, ... , vn }의 벡터들은
    • c1v1 + c2v2 + c3v3 + ... + cnvn = 0(영벡터)을 만족하는
    • 계수들이 c1 = 0, c2 = 0, c3 = 0 , ... , cn = 0이외에는 존재하지 않을 때,
    • 선형적으로 독립이라 한다.
