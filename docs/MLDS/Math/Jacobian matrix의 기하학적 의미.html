편미분 체인 룰자코비안의 의미(r, θ) 좌표계에서 (x, y)변환

 

편미분 체인 룰

|y1y2|=|abcd||x1x2|y1=ax1+bx2y2=cx1+dx1
  • 위와 같은 식이 있다.
  • y1과 y2를 t로 미분해보겠다.
dy1dt=∂y1∂x1dx1dt+∂y1∂x2dx2dtdy2dt=∂y2∂x1dx1dt+∂y2∂x2dx2dt

(단, y1, y2, x1, x2가 모두 t로 미분할 수 있을 때)

 

  • 따라서 아래의 수식이 성립한다.
dy1=∂y1∂x1dx1+∂y1∂x2dx2dy2=∂y2∂x1dx1+∂y2∂x2dx2|dy1dy2|=|∂y1∂x1∂y1∂x2∂y2∂x1∂y2∂x2||dx1dx2|

 

자코비안의 의미

 

 

 

J=|∂y1∂x1∂y1∂x2∂y2∂x1∂y2∂x2|

 

  • 자코비안 행렬의 의미를 생각해보자

  • 비선형 변환의 국소적 선형성을 뜻한다.

    • 무슨 얘기냐하면 비선형 변환도 국소적으로는 선형성을 뛴다는 것이다.

  • (이미지 출처 : https://angeloyeo.github.io/2020/07/24/Jacobian.html)

  • 결국 국소적인 단위에선 비선형변환도 어느정도의 선형성을 뛴다는 것이다.

  • 이걸 수학적으로 표현한 것

  • 좌표계 변환에 사용된다.

 

 

(r, θ) 좌표계에서 (x, y)변환

x=rcosθy=rsinθ
|dxdy|=|dxdrdxdθdydrdydθ||drdθ|

 

  • det(J)는 r이다.

  • 선형변환에서 행렬식의 의미를 생각해보자

    y = (y1, y2) , x = (x1 , x2)라고 했을 때

    y = Tx일 때

    det(T)*x1*x2 = y1 * y2 와 같다.

  • 이로 인해 좌표 변환에서 미소면적 r * dr * dθ = dx dy가 성립하는 것 같다.

    • 이걸 아마 중심극한정리 증명하는 거에서 봤던 수식 같다. 지금 당장은 스킵할까한다.
