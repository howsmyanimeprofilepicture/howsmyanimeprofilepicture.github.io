라그랑주 승수법

x, y가 g(x, y) = 0을 만족할 때,

함수 f(x, y)의 극값을 만드는 x, y는

∇f(x, y) = λ∇g(x, y)를 만족한다.

 

라그랑주 승수법의 기하학적 이해

  • 어떠한 제약 조건 아래서 함수의 극값을 찾을 때 사용한다.

  • x2 + y2 = 1 일때

  • x+y의 극대값, 극소값을 찾아보자.

  • 제약식을 xy좌표에 표현하면 이와 같다.

    • 제약식은 g(x, y) = x² + y² - 1 = 0으로 표현하겠다.

  • 근데 x와 y말고도 z축이 또 있다고 생각하자.

  • 우리가 극대화하고자 하는 값 x+y를 z라고 두자. 정리하면 z = f(x, y) = x+y라고 하자.

  • 그리고 각 z값별로 등위곡선을 그려주겠다.

  • 제약식(g(x, y) = 0) 같은 경우 따로 z값이 지정되지 않는 x와 y만으로 이루어진 방정식이다.

    • 따라서 z축이 있는 3차원 공간으로 생각할 때 원기둥형태의 벽이 세워진 것과 같다.
    • z를 몇으로 설정해서 등위곡선을 그리던 g(x, y) = x² + y² - 1 = 0은 똑같은 모양일 것이다.
    • 제약식은 일종의 조건이고, g(x, y) = 0이라는 조건에 z에 관한 내용은 단 하나도 없다!! 따라서 z값이 몇이 되든 g(x, y) = 0이라는 조건은 동일할 것이다.

  • 이와 같이 z=√2 일 때의 등위선이 제약식이 만든 원(g(x, y) = x² + y² - 1 = 0)과 접함을 알 수 있다.

  • 동시에 이 지점에서 f(x, y)가 극대값을 갖는다는 것도 알 수 다.

  • 이 때 f(x, y)의 등위선과 제약식의 공통접선이 동일하다는 것을 알 수 있고, 이 말은 즉슨 해당 지점에서 f(x, y)의 그라디언트 벡터와 제약식(g(x, y) = 0)의 그라디언트 벡터가 평행하다는 것을 의미한다.

  • (참고로 그라디언트 벡터는 접선과 수직이다.)

  • 이로 인해 라그랑주 승수법이 성립할 수 있는 것이다.

라그랑주 함수의 유도

  • 제약식이 g(x, y) = 0이고, 극대화하려는 값이 f(x, y)라면, 라그랑주 함수는 다음과 같이 표현할 수 있다.

    L(x, y, λ) = f(x) - λg(x)

    이 때 ∇L(x, y, λ)를 영벡터로 만드는 x, y, λ가 극값이다.

    이말은 즉슨 ∇f(x, y) = λ*∇g(x, y) 과 g(x) = 0을 동시에 만족하라는 이야기가 된다.
