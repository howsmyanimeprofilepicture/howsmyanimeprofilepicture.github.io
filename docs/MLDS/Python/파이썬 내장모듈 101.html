os

os.path.isdir(path)

  • path가 존재하는 디렉토리일 경우 True가 반환됨

os.path.isfile(path)

  • path가 존재하는 파일이면 True 반환함

os.makedirs('./first/nested1/nested2', exist_ok=True)

  • exist_ok를 True로 하면, 기존에 해당 디렉토리가 존재해도 에러없이 패스한다.
  • os.mkdir()은 중첩된 디렉토리 생성(./a/b/c/d/e 같은)이 안 되는 것으로 안다.

os.listdir(path)

  • 지정한 디렉토리에 있는 모든 파일과 디렉토리 이름의 리스트를 리턴한다.
  • Paramete를 생략하면 자동으로 현재 디렉토리를 기준으로 리턴

os.getcwd()

  • cwd == Client Working Directory
  • 현재 디렉토리를 리턴한다.

os.symlink(src, dst, target_is_directory=False, ***, dir_fd=None)

  • src를 가르키는 dst라는 이름의 심볼릭 링크를 만든다.

    • 심볼릭 링크(symbolic link)란? - windows 운영체제로 치면 바로가기와 비슷하다.
    • 해당 src로 가는 링크이다. src는 폴더, 파일, 웹 모두 가능하다.

  •  

 

shutil

shutil.move(previous_dir, new_dir)

  • previous_dir 파일을 new_dir 파일로 이동시킴.

  • shutil.move('./a/a.jpg', './b/b1/b.jpg')

    • 대충 ./a/a.jpg 파일을 ./b/b1/ 폴더로 보낸 후 이름을 b.jpg로 바꾼 것으로 이해해도 무방

 

shutil.rmtree(dir)

  • dir이라는 폴더를 지운다. (하위 파일 폴더 전부 삭제)
  • 파일이 아닌 디렉토리(폴더)를 인자로 전달해야 한다.

 

 

sys

sys.path.insert

x
import sys
sys.path.insert(0, './a/b/c')

  • 환경변수를 설정한다.

  • 위 예시에선 c폴더를 환경변수로 설정한다는 뜻이다.

  • 이렇게 환경변수로 설정하면 그 파일 안의 파이썬 파일들을 간편하게 import할 수 있다.

  • 만약 ./a/b/c 폴더에 있는 maru.py 파일에 있는 mungmung함수를 사용하고 싶다면

    • xxxxxxxxxx
      import sys
      sys.path.insert(0, './a/b/c')
      from maru import mungmung

  • 만약 2번째 인자로 심볼릭 링크를 넣는다면 심볼링 링크가 참조하는 src를 환경변수로 삼는다.
