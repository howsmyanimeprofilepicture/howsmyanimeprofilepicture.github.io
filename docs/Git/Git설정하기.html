  • -e (혹은 --edit)은 에디터와 관련된 것이고, (여기서 vi를 쓸것이냐, vscode를 쓸것이냐를 고를 수 있음.)

  • --global은 전역적인 설정값들을 관리할 수 있다.

     

 

2. 많이 쓰는 git 설정들

  • xxxxxxxxxx
    $ git config -e --global
    • 에디터를 설정할 떄 쓴다. vi를 쓸지, vscode를 쓸지를 정할 때
  • x
    $ git config --global user.name "howsmyanimeprofilepicture"
    $ git config --global user.email "howsmyanimeprofilepicture@gmail.com"

    이렇게 user.name이나 email을 설정할 수도 있다.
