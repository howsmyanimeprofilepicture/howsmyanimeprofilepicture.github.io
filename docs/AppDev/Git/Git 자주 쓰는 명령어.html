1. 자주 쓰는 git 커맨드

  • git init

  • git remote add origin https://github.com/GITHUB_ACCOUNT

    • 깃헙과 로컬저장소(origin)를 연동한다.

  • git push -u origin master

    • 로컬저장소의 origin을 깃허브의 master브랜치로 푸쉬한다.
    • -u옵션을 주면 origin과 원격저장소의 master 브랜치를 자동으로 연결한다.
    • git push git pull 만 입력하면 자동으로 origin이 master로 push되고, pull된다.

 

  • git clone https://github.com/GITHUB_ACCOUNT

    • 저장소를 그대로 클론한다.

  • git remote rm origin

    • origin을 삭제한다.

  • git add *

  • git commit -m '할 말 '
