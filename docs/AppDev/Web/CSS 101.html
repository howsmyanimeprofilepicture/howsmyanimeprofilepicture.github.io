@import

참조 https://developer.mozilla.org/ko/docs/Web/CSS/@import

 

외부 CSS파일을 CSS파일 내에서 import할 수 있음.

@import url("fineprint.css") print;
@import url("bluish.css") speech;
@import 'custom.css';
@import url("chrome://communicator/skin/");
@import "common.css" screen;
@import url('landscape.css') screen and (orientation:landscape);

 

 

 

overflow

참조 : https://developer.mozilla.org/ko/docs/Web/CSS/overflow

  • overflow CSS 단축 속성은 요소의 콘텐츠가 너무 커서 요소의 블록 서식 맥락에 맞출 수 없을 때의 처리법을 지정합니다. overflow-x (en-US), overflow-y (en-US)의 값을 설정합니다.

 

 

 

@media

xxxxxxxxxx
.box {
    color: blue;
}


@media screen and (max-width: 480px) {
    /* 뷰포트 480px 이하는 여기 기준으로 동작 */
    .box {
        color: red;
    }
}


@media screen and (min-width: 1280px) {
    /* 뷰포트 1280px 이상은 여기 기준으로 동작 */
    .box {
        color: yellow;
    }
}
